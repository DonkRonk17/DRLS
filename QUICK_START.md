### **Save Your Design**
1. Click "Export to DRGUI" when satisfied
2. Choose your profile name
3. The system automatically:
   - Creates DRGUI addon profile
   - Copies to your WoW AddOns folder
   - Makes it ready to load in-game

### **Load In-Game**
1. Start World of Warcraft
2. Type: `/drgui profile load [your-profile-name]`
3. Enjoy your perfect AI-designed UI!

## ðŸ†˜ **Troubleshooting**

### **Launcher Issues**
```
Problem: "Python not found"
Solution: Install Python 3.8+ from python.org

Problem: "Dependencies failed to install"
Solution: Run PowerShell as Administrator and try again

Problem: "Virtual environment error"
Solution: Delete the 'venv' folder and re-run launcher
```

### **Application Issues**
```
Problem: AI not responding
Solution: Check internet connection or enable offline mode

Problem: Simulation is laggy
Solution: Reduce quality in Settings > Performance

Problem: Can't export to WoW
Solution: Verify WoW installation path in Settings > Export
```
